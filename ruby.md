# Ruby
`irb`
: REPL shell
## Functions
`def`
: begin a function definition
`end`
: end a function definition
`require {package}`
: import {package}, or 'gem'
`puts(*obj)`
: write given objects to `ios`; writes newline after any that do not already have one
`print(*obj)`
: write given objects to `ios`, with no newline
### Array methods
`array.sort()`
: sort in-place
