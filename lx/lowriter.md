# lowriter
`lowriter` is a command-line utility installed with LibreOffice Writer.
#### Convert a single file to PDF
```sh
lowriter --convert-to pdf filename.doc
```
#### Convert a batch of files using globbing
```sh
lowriter --convert-to pdf *.docx
```
