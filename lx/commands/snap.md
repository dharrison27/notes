### snap
Disable snap {pkg}
```sh
snap disable package
```
Enable disabled snap {pkg}
```sh
snap enable package
```
Display information about {package}
```sh
snap info --verbose package
```
Install snap {pkg}
```sh
snap install package
```
Display logs of snap {pkg}
```sh
snap logs package
```
Check for snap updates
```sh
snap refresh
```
Uninstall snap {pkg}
```sh
snap remove package
```
