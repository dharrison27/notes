### `ps`
Display processes in a tree-like display illustrating parent-child relationships
```sh
ps -f
ps --forest
```
Show system processes
```sh
ps ax
ps -e
```
Display full listing of processes
```sh
ps u
ps -f
```
Display user processes
```sh
ps xG
ps -a
```
Display SELinux contexts for processes
```sh
ps auxZ
```
